Constructor for the CrystalExtractor class.

Initializes some custom private data (not present in base class) for use when calculating some features. Sets the sessionID feature. Further, initializes all Q&A features to -1, representing unanswered questions.

Parameters
session_idThe id number for the session whose data is being processed by this extractor instance.
game_tableA data structure containing information on how the db table assiciated with this game is structured.
game_schemaA dictionary that defines how the game data itself is structured.
