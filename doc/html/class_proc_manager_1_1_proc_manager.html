Simply stores some data for use later, including the type of extractor to use.

Parameters

game_tableA data structure containing information on how the db table assiciated with the given game is structured.
game_schemaA dictionary that defines how the game data itself is structured.
proc_csv_fileThe output file, to which we'll write the processed feature data.
